# 需求分析草稿

## 1. 需求背景与目标

5 月推广节奏中，5.15 版本需要提审一批运营活动相关能力，重点支持“浴室歌王争霸赛”和“丑东西大赛”。当前 `requirement.md` 主要写明了通用资源位、消息中心活动入口、官号认证、信息流排序强插，以及“丑东西大赛”的 H5 和活动玩法配套能力；“浴室歌王争霸赛”仅作为背景目标出现，具体玩法规则未在正文展开。

本次目标是完成通用资源位与运营活动能力搭建，使运营侧能够配置开屏资源位、消息中心活动露出、官号认证和内容强插，并支撑活动 H5、活动聚合、点赞特效、短链跳转和头像框奖励等能力。

当前需求中包含“图片或视频素材”“全览图”“头图”等图片/素材表述，但本草稿未分析 PDF 或图片二进制内容，只基于 `requirement.md` 正文生成。图片或原型图中若存在未转写的字段、按钮、状态、样式或交互规则，需要人工补充到正文后再复审。

## 2. 业务范围

**范围内**

- APP 启动页新增开屏资源位，支持配置图片或视频素材。
- 开屏资源位冷启动触发、本地缓存命中才展示、每日一次展示、3 秒倒计时或跳过进入首页。
- 点击开屏广告非跳过区域，跳转到 APP 内指定帖子详情页或站内 H5 页面；从落地页返回后进入首页，不再次展示开屏广告。
- 资源位后台新增配置逻辑，同一生效时间段仅支持配置一个活动。
- APP 消息中心活动板块展示最新活动 list，按时间倒序展示，并通过数字角标、小红点和当日首次进入 APP 展开提示引导参与。
- 指定官方号增加认证标识，头像展示位置可见，点击后浮层提示“交子立方官方认证账号”，样式为 V 字样式。
- 指定官方号发布内容可将指定内容 ID 强插到信息流前两位，并受 24 小时和自然日限制。
- 丑东西大赛 H5 页面包含头图、活动时间、活动简介、排行榜、广场、活动奖励和参与入口。
- 丑东西大赛相关帖子点赞后展示“丑东西”特效。
- 话题详情页新增按帖子热度排序能力。
- 发帖与评论区支持发布短链，点击跳转到 H5 活动详情页。
- 新增头像框类型，支持配置佩戴时长和样式。

**范围外或未明确**

- 浴室歌王争霸赛的具体玩法、页面模块、奖励和参与规则未在正文展开。
- 活动后台完整字段、审批流、素材上传限制、素材审核、素材缓存下发策略未明确。
- 接口路径、请求参数、响应字段、错误码、埋点字段、消息推送策略未提供。
- 图片/视频素材的尺寸、格式、大小、横竖屏适配、静音/播放控制和失败兜底素材未明确。
- 丑东西大赛“符合活动规则的帖子”判定条件未明确。
- 排行榜计算口径、热度算法、奖励发放规则和头像框发放方式未明确。

## 4. 主流程拆解

1. 运营在资源位配置后台配置开屏活动素材，选择图片或视频、落地页、活动生效时间等信息。
2. 后台保存配置，并保证同一生效时间段仅有一个活动配置生效。
3. APP 冷启动时检查是否符合开屏广告触发条件，并检查本地是否有对应开屏素材缓存。
4. 缓存命中且当日未展示过时，APP 展示开屏素材和倒计时。
5. 用户等待倒计时结束或点击跳过，直接进入 APP 首页。
6. 用户点击广告非跳过区域，跳转到 APP 内帖子详情页或站内 H5 页面。
7. 用户从广告落地页返回后，直接进入 APP 首页，本次不再次展示开屏广告。
8. 用户进入消息中心时，活动板块按时间倒序展示最新活动 list，并展示数字角标、小红点或当日首次进入 APP 的活动提示。
9. 指定官方号在 feed、客态个人页、我的页、活动页等头像位置展示 V 样式认证标识；用户点击标识后展示官方认证浮层。
10. 指定官方号发布的内容 ID 按强插规则进入信息流前两位，并受 24 小时和自然日限制。
11. 用户进入丑东西大赛 H5，查看头图、活动时间、简介、排行榜、广场、奖励和参与入口。
12. 符合活动规则的帖子被聚合到活动广场；用户点赞符合条件的帖子后展示“丑东西”特效。
13. 用户在话题详情页可按帖子热度排序浏览活动内容。
14. 用户在发帖或评论区发布短链，其他用户点击短链跳转到 H5 活动详情页。
15. 活动奖励支持头像框类型，头像框可配置样式和佩戴时长，具体发放规则待确认。

## 5. 分支流程与异常流程

| 场景 | 触发条件 | 预期处理 |
|---|---|---|
| 开屏素材未缓存 | APP 冷启动但本地无对应素材缓存 | 不展示开屏广告，直接进入首页；不得阻塞启动 |
| 非冷启动唤醒 | 前后台切换、系统推送唤醒、快捷入口启动、崩溃重启等 | 不触发展示开屏广告 |
| 当日已展示过开屏 | 同一自然日再次冷启动 | 不再次展示开屏广告 |
| 倒计时结束 | 开屏素材展示到 3 秒倒计时结束 | 自动进入 APP 首页 |
| 用户点击跳过 | 倒计时未结束时点击跳过按钮 | 立即进入 APP 首页，不触发广告落地页 |
| 用户点击广告区域 | 点击除跳过按钮外任意广告区域 | 跳转到指定帖子详情页或站内 H5 页面 |
| 落地页返回 | 用户从广告落地页返回 | 直接进入 APP 首页，不再次展示开屏广告 |
| 配置时间冲突 | 后台同一生效时间段存在多个活动配置 | 应阻断或提示冲突；具体提示和保存规则待确认 |
| 消息中心无活动 | 无最新活动或活动均已查看 | 不展示活动红点或展开提示；空态文案待确认 |
| 当日首次进入 APP 且有未查看活动 | 用户首次进入 APP 且活动未查看 | 展开提示参与活动，展示规则和关闭后状态待确认 |
| 官方认证账号未配置 | 非指定账号或配置失效 | 不展示认证标识 |
| 点击官方认证标识 | 用户点击 V 标识 | 展示“交子立方官方认证账号”浮层 |
| 强插内容已超每日限制 | 单个自然日已强插一条内容 | 不再强插新增内容或按待确认策略处理 |
| 强插内容超过 24 小时 | 同一内容 ID 超过 24 小时 | 是否继续强插一次及强插位置需确认 |
| 活动帖子不符合规则 | 帖子未满足丑东西大赛规则 | 不进入活动广场，不触发点赞特效；规则待确认 |
| H5 短链失效或非法 | 发帖/评论区短链无效、过期或不安全 | 应拦截或展示错误提示，避免跳转风险 |
| 头像框配置缺失 | 奖励配置缺少样式或佩戴时长 | 应阻断配置或不发放，具体校验规则待确认 |

## 10. 风险点与影响面

| 风险点 | 影响面 | 建议处理 |
|---|---|---|
| 倒计时时长存在 5s 待定与 3s 口径 | 开屏体验、用例预期、提审验收 | 产品确认最终时长，并统一前端/后台配置 |
| 开屏本地缓存命中才展示 | 活动曝光可能不稳定 | 明确缓存下发时机、缓存过期和弱网兜底 |
| 每日一次展示维度不明 | 多账号、多设备、跨天测试可能偏差 | 确认自然日按设备、账号、用户还是安装实例 |
| 同一生效时间段仅一个活动 | 后台配置冲突和运营排期风险 | 后台保存时强校验并提示冲突活动 |
| 消息中心角标/红点/展开提示规则不清 | 用户打扰、漏提醒或重复提醒 | 明确未查看状态、清除时机和每日首次口径 |
| 官号认证覆盖位置多 | feed、个人页、我的页、活动页展示不一致 | 建立页面清单和统一组件回归 |
| 信息流强插影响排序 | 可能影响内容公平、广告位、置顶逻辑 | 确认排序优先级和曝光去重策略 |
| 24 小时与自然日限制可能冲突 | 强插次数计算和跨天行为不一致 | 明确两类限制的优先级和重置时间 |
| 丑东西帖子判定规则缺失 | 广场、排行榜、点赞特效无法准确验证 | 产品补充活动帖规则、话题/标签/审核条件 |
| 热度排序算法缺失 | 话题详情页排序结果无法验收 | 补充热度字段、权重和刷新周期 |
| 短链发布存在安全风险 | H5 跳转、内容审核、反垃圾风险 | 增加短链白名单、过期和内容审核策略 |
| 头像框奖励规则不完整 | 奖励发放、佩戴过期和资产一致性风险 | 补充奖励获得、领取、佩戴和回收流程 |
| 图片/视频素材规则不完整 | 素材格式、尺寸、播放失败无法验收 | 补充素材规格、封面、播放控制和失败兜底 |
| 未提供接口文档 | 无法验证字段、错误码和服务状态 | 评审前补充 api-doc 或接口字段清单 |
