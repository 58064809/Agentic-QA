# 需求分析草稿

## 1. 需求背景与目标

**需求来源**：`prd/5月活动玩法/input/requirement.md`

**背景**：5月活动期间，结合“脆皮打工人”的热点话题，通过H5活动“脆皮打工人养护中心”进行用户互动和传播，旨在提升平台活跃度、用户参与度以及新用户增长。

**业务目标**：
- **核心目标**：吸引用户参与“脆皮打工人”测试，生成个性化诊断书并分享，实现活动传播。
- **增长目标**：通过邀请好友裂变玩法，带动新用户注册和参与。
- **留存目标**：将活动流量导流至平台职场圈、治愈圈、专业咨询板块，形成“活动引流→长期留存”的闭环。
- **互动目标**：鼓励用户在活动话题区发布和评论内容，形成社区氛围。

## 2. 业务范围

**范围内**：
- **核心活动**：“脆皮打工人养护中心”H5活动页面。
    - 用户参与包含10道选择题的“脆皮测试”。
    - 系统根据答案自动计算评分并生成专属“电子诊断书”。
    - 诊断书支持保存为图片（带平台logo）和分享。
- **社区互动**：
    - 活动专属话题区，用户可发布诊断书并配文，其他用户可评论“会诊”。
    - 官方运营号带头评论互动。
- **裂变拉新**：
    - 老用户生成专属邀请链接/海报。
    - 新用户通过链接/二维码注册APP并完成测试、发布内容后，老用户获得奖励。
- **导流闭环**：
    - 活动页设置“专业养护通道”入口，引流至专业圈。
    - 活动话题区置顶“职场摸鱼圈”“治愈放松圈”“脆皮养护交流圈”入口。
    - 主理人发起线上茶话会活动。
- **奖励体系**：
    - 完成测试即可获得“脆皮打工人”专属头像框（有效期15天）。
    - 评论区随机抽取50名用户获得“终极养护礼包”。

## 4. 主流程拆解

**主流程：用户参与脆皮测试并分享**

1.  **用户入口**：用户从H5活动宣传页或分享链接进入活动。
2.  **参与测试**：用户依次回答10道选择题。
3.  **提交评分**：用户完成所有题目后，系统自动计算得分。
4.  **结果展示**：系统根据得分（10-15分/16-25分/26-40分）显示对应的诊断结果、症状和医嘱。
5.  **生成诊断书**：用户触发生成专属电子诊断书（图片形式，带平台logo）。
6.  **用户操作**：用户可选择：
    - **保存/分享**：将诊断书图片保存到本地或分享到社交媒体。
    - **发布内容**：在活动话题区发布诊断书和配文。
    - **查看专业建议**：进入“专业养护通道”。
    - **邀请好友**：生成邀请链接/海报进行裂变。
7.  **奖励发放**：用户完成测试后立即获得“脆皮打工人”专属头像框。

**裂变主流程**：
1.  **老用户生成邀请**：老用户在活动页生成专属邀请链接或海报。
2.  **新用户参与**：新用户通过链接/海报二维码注册APP，并完成测试+发布1条互动内容。
3.  **奖励结算**：系统判定为有效邀请，老用户获得“脆皮养护礼包”或“一次免费专业养护通道活动”。
4.  **闭环**：新用户成为平台用户，可能继续参与活动。
