# 需求分析草稿

## 1. 需求背景与目标

- **需求背景**：基于“脆皮打工人”的社会热点话题，通过H5互动测试和社区讨论，吸引职场人群参与，增强平台用户活跃度和社交互动。
- **业务目标**：
  - 用户参与脆皮测试并生成诊断书。
  - 通过诊断书分享和社区互动实现社交裂变，带动新用户注册。
  - 将活动流量导流至平台的“职场圈”“治愈圈”“专业咨询”等核心模块，提升长期留存。
- **上线范围**：H5活动页面（含测试、诊断书生成、社区话题区、裂变邀请、导流入口）。
- **假设**：假设所有测试题目均在H5页面内一次性展示或分步展示（需确认设计图）。

## 2. 业务范围

### 范围内
- 用户在线完成10道脆皮测试题目。
- 系统根据选项自动计算总分，并生成对应等级的诊断书（文字内容+配图）。
- 用户可保存、分享诊断书图片。
- 用户可发布诊断书到活动话题区，并配文参与互动。
- 其他用户可在话题区内评论、互动（“会诊”）。
- 老用户邀请新用户参与活动（裂变玩法）。
- 活动页面设置导流入口，链接至平台的职场圈、治愈圈、专业咨询板块。
- 提供奖励：参与者获得限定头像框，评论区抽奖获得养护礼包。

## 4. 主流程拆解

### 流程一：用户参与测试并生成诊断书
1. 用户进入H5活动页，看到宣传物料（图片+文案）。
2. 用户开始答题，依次回答10道题目（每题4个选项）。
3. 用户提交答案。
4. 系统根据A/B/C/D分别计4/3/2/1分，计算总分。
5. 系统根据总分匹配等级（轻度/中度/重度）及对应的诊断结果、玩梗文案、症状图片列表。
6. 系统生成诊断书图片（含用户信息、等级、症状、医嘱、平台logo）。
7. 用户可查看、保存、分享诊断书图片。

### 流程二：用户在话题区互动
1. 用户生成诊断书后，可选择跳转至活动话题区。
2. 用户在话题区发布内容（配文+诊断书图片）。
3. 其他用户浏览帖子，并在评论区“会诊”（发表评论）。
4. 官方运营号带头评论，引导用户参与。

### 流程三：邀请裂变
1. 老用户在活动页生成专属邀请链接/海报。
2. 新用户通过链接/海报二维码注册APP（若未注册）。
3. 新用户完成脆皮测试并发布1条互动内容。
4. 系统判定邀请有效，向老用户发放奖励（养护礼包或免费养护通道活动）。
5. 新用户也可能获得新人奖励（未明确）。

### 流程四：导流至平台功能
1. 活动页设置“专业养护通道”入口，点击跳转至平台主理人专业圈。
2. 活动话题区置顶“职场摸鱼圈”等圈子入口，点击跳转至对应圈子。
