# 需求分析草稿 – 5月活动玩法：脆皮打工人养护中心

## 1. 需求背景与目标

**需求原文**：H5活动宣传页，Slogan“拒绝内卷，在线会诊”，面向“脆皮打工人”群体，通过趣味测试、诊断书生成、社交裂变和社区互动，提升平台活跃度并导流至职场圈、治愈圈和专业咨询板块。

**业务目标**：
- 用户参与脆皮测试并生成专属诊断书（可保存/分享/发布到话题区）
- 通过裂变邀请带动新用户注册与互动
- 将活动流量导流至平台核心圈子（职场摸鱼圈、治愈放松圈、脆皮养护交流圈）和专业主理人板块
- 降低参与门槛（头像框参与奖），激励深度互动（评论区抽奖）

**上线范围**：H5活动页（猜测嵌入APP或微信H5），未明确系统载体，**假设**为APP内嵌H5或独立H5页面。

## 2. 业务范围

### 范围内

| 模块 | 说明 |
|------|------|
| 脆皮测试 | 10道选择题，每道4个选项，选项分值A=4,B=3,C=2,D=1 |
| 评分与等级划分 | AI自动计算总分，10-15轻度脆皮，16-25中度脆皮，26-40重度脆皮 |
| 诊断书生成 | 根据等级生成固定文案（诊断结果、症状、医嘱），可生成图片（含平台logo），支持保存/分享 |
| 话题区互动 | 用户可将诊断书发布到活动话题区，配文“求会诊”“求药方”；其他用户可评论“会诊” |
| 裂变邀请 | 老用户生成专属邀请链接/海报，新用户通过链接/海报二维码注册APP并参与活动（完成测试+发布1条互动内容）视为有效邀请 |
| 奖励发放 | 完成测试即得“脆皮打工人”专属头像框（有效期15天）；评论区随机抽取50名获终极养护礼包 |
| 导流入口 | 活动页设置“专业养护通道”，话题区置顶圈子入口（职场摸鱼圈、治愈放松圈、脆皮养护交流圈），主理人联动（线上茶话会） |

|------|------|

| 定时红包活动 | 仅提及图片链接，无具体需求描述，本期不分析 |
| 后台管理 | 奖励发放、评论审核、数据统计等后台功能未提及，**假设**另有后台系统支持 |

## 3. 角色与权限

| 角色 | 说明 | 权限 |
|------|------|------|
| 用户（脆皮打工人） | 参与活动的主要角色，包括新用户和老用户 | 答题、生成诊断书、保存/分享、发布话题、评论、邀请好友、领取奖励（头像框/礼包） |
| 未登录用户（游客） | 是否可浏览活动页但参与活动？**待确认** | **假设**未登录用户不能参与测试和互动，需先登录/注册 |
| 新用户（被邀请者） | 通过邀请链接/海报注册APP并参与活动的用户 | 完成测试+发布1条互动内容后，老用户（邀请者）获得奖励 |
| 官方运营 | 带头评论，用玩梗语气“会诊” | 发布评论、置顶帖子、管理话题区 |
| 主理人（平台专业圈） | 发布专业小建议，发起线上茶话会 | 发布内容、创建社群活动（权限由平台权限体系决定） |

**权限差异**：未提及用户等级或收费用户特权，**假设**所有用户均可平等参与。

## 4. 主流程拆解

### 主成功路径（用户参与测试并发布话题）

1. 用户进入活动H5页面（需登录？**待确认**）
2. 用户依次回答10道测试题（每题选择A/B/C/D）
3. 用户提交答案，系统调用AI评分接口
4. 系统计算总分并映射到等级（轻度/中度/重度）
5. 系统根据等级展示专属诊断书（含诊断结果、症状、医嘱）
6. 用户可选择：【保存为图片（含平台logo）】、【分享到社交平台】、【发布到活动话题区】
7. （可选）用户发布到话题区后，其他用户可评论“会诊”
8. 用户完成测试即自动获得头像框（有效期15天）

### 裂变邀请主流程

1. 老用户进入活动页，点击“邀请好友”生成专属邀请链接/海报
2. 老用户将链接/海报分享给新用户
3. 新用户点击链接/扫描二维码，跳转APP注册页（或直接注册流程）
4. 新用户注册成功后，进入活动页完成测试并发布至少1条互动内容
5. 系统判定邀请有效，老用户获得“脆皮养护礼包”或“一次免费专业养护通道活动”

### 专业养护通道导流

1. 用户在活动页点击“专业养护通道”入口
2. 跳转到平台专业圈（心理/养生/职场）主理人页面
3. 用户可关注主理人、查看建议、咨询问题

## 5. 分支流程与异常流程

| 流程类型 | 描述 |
|---------|------|
| 分支-诊断书不发布 | 用户仅保存/分享图片，不发布到话题区，头像框仍发放 |
| 分支-邀请链接失效 | 判断链接是否过期或达到邀请上限，**假设**有有效期或使用次数限制 |
| 分支-新用户未完成互动 | 若新用户只完成测试但未发布互动，则邀请无效 |
| 异常-答题中断 | 用户答题中途退出，再次进入时是否保留已选项？**待确认** |
| 异常-评分接口超时 | 用户提交后长时间等待，需提供加载提示和重试机制 |
| 异常-诊断书生成失败 | 图片生成服务不可用，应提供纯文本诊断书作为降级 |
| 异常-头像框发放失败 | 如发放失败，是否提供重试或人工补发？**待确认** |
| 异常-分享失败 | 平台分享接口受限，需提示用户手动保存 |
| 异常-邀请关系绑定错误 | 新用户注册时未正确关联邀请人，**假设**通过invite_code参数绑定 |
| 异常-奖品库存不足 | 终极养护礼包限量50份，若中奖用户超过库存，**假设**优先中奖先得，需确认抽奖逻辑 |

```

邀请状态 → 已完成 → 触发老用户奖励发放
```
