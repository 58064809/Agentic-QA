### 未覆盖范围说明

- **性能测试**：未覆盖高并发场景（如万人同时测试、诊断书生成并发），因未提供性能要求，待确认。
- **安全测试**：未覆盖XSS/CSRF攻击、SQL注入等，因活动H5页的安全要求未提供，待确认。
- **弱网测试**：未覆盖2G/3G网络下的活动体验，仅覆盖了网络断开场景，建议在性能测试阶段补充。
- **中断测试**：未覆盖电话打断、低电量关机等移动端中断场景，因未提供相关要求。
- **图片内容测试**：由于图片素材未提供，未覆盖诊断书图片、邀请海报图片、症状图片的UI测试，需待UI交付后补充。

1. **接口文档缺失**：所有API接口（测试提交、诊断书生成、邀请链生成、奖励发放）未提供，用例中的接口响应和字段为假设，需等待开发提供接口文档后补充。
2. **图片素材未提供**：诊断书模板、症状配图（轻度/中度/重度各16张）、邀请海报模板、头像框图片均未提供，无法测试图片内容和视觉交互，需等待设计交付。
3. **活动载体未确认**：活动是APP内嵌H5还是独立H5页面？iOS/Android最低版本支持？WebView版本要求？这些影响兼容性测试范围。
