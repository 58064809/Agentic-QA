## 4. 主流程拆解

**主流程：用户参与脆皮测试并分享**

1.  **用户入口**：用户从H5活动宣传页或分享链接进入活动。
2.  **参与测试**：用户依次回答10道选择题。
3.  **提交评分**：用户完成所有题目后，系统自动计算得分。
4.  **结果展示**：系统根据得分（10-15分/16-25分/26-40分）显示对应的诊断结果、症状和医嘱。
5.  **生成诊断书**：用户触发生成专属电子诊断书（图片形式，带平台logo）。
6.  **用户操作**：用户可选择：
    - **保存/分享**：将诊断书图片保存到本地或分享到社交媒体。
    - **发布内容**：在活动话题区发布诊断书和配文。
    - **查看专业建议**：进入“专业养护通道”。
    - **邀请好友**：生成邀请链接/海报进行裂变。
7.  **奖励发放**：用户完成测试后立即获得“脆皮打工人”专属头像框。

**裂变主流程**：
1.  **老用户生成邀请**：老用户在活动页生成专属邀请链接或海报。
2.  **新用户参与**：新用户通过链接/海报二维码注册APP，并完成测试+发布1条互动内容。
3.  **奖励结算**：系统判定为有效邀请，老用户获得“脆皮养护礼包”或“一次免费专业养护通道活动”。
4.  **闭环**：新用户成为平台用户，可能继续参与活动。
