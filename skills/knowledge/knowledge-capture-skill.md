# 知识沉淀 Skill

## 元数据

- Skill ID: S10
- 第一版必装: 是
- 适用阶段: Runtime QA Workflow

## 作用

把确认后的规则、用例、缺陷经验沉淀到知识库

## 输入

人工确认后的分析、用例、缺陷、报告

## 输出

知识库规则、历史用例索引、缺陷经验

## 执行步骤

- 只沉淀人工确认或审核通过的内容
- 提取可复用业务规则、风险模式、测试数据组合和缺陷根因
- 写入 knowledge 或 RAG 索引，保留来源和适用范围
- 避免把未确认假设沉淀为事实

## 质量门

- 输出必须可追溯到 PRD、规则、历史知识或人工确认项。
- 不确定内容必须标记“待确认”“待补充”或“假设”。
- 不得输出纯模板、占位内容或未经确认的正式结论。
