# RAG 检索 Skill

## 元数据

- Skill ID: S3
- 第一版必装: 是
- 适用阶段: Runtime QA Workflow

## 作用

从知识库、历史用例、历史缺陷、规则文档中召回相关内容

## 输入

需求文本、分析草稿、知识库、历史 QA 产物

## 输出

相关规则、历史用例、历史缺陷、风险提示

## 执行步骤

- 构造业务关键词、模块关键词、风险关键词查询
- 优先召回项目规则、同类历史用例、同类缺陷
- 过滤低相关、重复、过期或无法追溯内容
- 把召回结果标注来源后放入上下文

## 质量门

- 输出必须可追溯到 PRD、规则、历史知识或人工确认项。
- 不确定内容必须标记“待确认”“待补充”或“假设”。
- 不得输出纯模板、占位内容或未经确认的正式结论。
