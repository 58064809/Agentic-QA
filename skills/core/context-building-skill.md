# 上下文构建 Skill

## 元数据

- Skill ID: S2
- 第一版必装: 是
- 适用阶段: Runtime QA Workflow

## 作用

整理用户输入 + 项目知识 + 历史记忆，形成完整上下文

## 输入

用户自然语言、PRD 工作区、rules、prompts、knowledge、skills

## 输出

可供 LLM/Workflow 使用的上下文包

## 执行步骤

- 合并用户意图、目标 PRD 路径和任务类型
- 加载规则、模板、工作流、项目知识和目标 PRD 文件
- 去除重复上下文，避免旧草稿与本轮草稿重复塞入 Prompt
- 记录缺失文件、图片限制和待确认上下文

## 质量门

- 输出必须可追溯到 PRD、规则、历史知识或人工确认项。
- 不确定内容必须标记“待确认”“待补充”或“假设”。
- 不得输出纯模板、占位内容或未经确认的正式结论。
