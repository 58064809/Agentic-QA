# 测试方法选择 Skill

## 元数据

- Skill ID: S6
- 第一版必装: 是
- 适用阶段: Runtime QA Workflow

## 作用

自动判断该用边界值、等价类、判定表、状态迁移、场景法等

## 输入

测试范围、业务规则、字段、状态、风险清单

## 输出

测试方法选择矩阵

## 执行步骤

- 字段输入类优先选择等价类和边界值
- 多条件业务规则优先选择判定表/因果图
- 状态变化类优先选择状态迁移
- 端到端业务活动优先选择场景法和风险测试

## 质量门

- 输出必须可追溯到 PRD、规则、历史知识或人工确认项。
- 不确定内容必须标记“待确认”“待补充”或“假设”。
- 不得输出纯模板、占位内容或未经确认的正式结论。
